UnitId crown::World::unit_by_name ( StringId32  name)

Returns the unit with the given Level Editor name or UNIT_INVALID if no such unit exists.

If there are multiple units with the same name, a random one will be returned.

void crown::World::disable_unit_callbacks ( )

After this call, units will no longer receive spawned(), unspawned() or any other callback.

This is used only by the editor to prevent user logic to interfere with editor's assumptions.
